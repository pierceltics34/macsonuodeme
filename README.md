# macsonuodemeli.com

Premium, sade ve güven veren web sitesi iskeleti.

## Teknolojiler

- Next.js 14 (App Router)
- TypeScript
- TailwindCSS
- Plus Jakarta Sans (Google Fonts)

## Kurulum

```bash
npm install
# veya
yarn install
# veya
pnpm install
```

## Geliştirme

```bash
npm run dev
# veya
yarn dev
# veya
pnpm dev
```

Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açın.

## Build

```bash
npm run build
npm start
```

## Sayfa Yapısı

- `/` - Anasayfa
- `/iletisim` - İletişim
- `/hakkimizda` - Hakkımızda
- `/referanslar` - Referanslar
- `/kuponlarimiz` - Kuponlarımız

## Tasarım Sistemi

- **Renkler:**
  - Primary (Yeşil): #16A34A
  - Secondary (Turuncu): #F97316
  - Background: #FFFFFF
  - Card Background: #F7F7F7 / #FAFAFA

- **Font:** Plus Jakarta Sans (tek font, tüm sitede)
- **Border Radius:** 16px
- **Max Width:** 1120px
